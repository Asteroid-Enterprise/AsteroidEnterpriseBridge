# AsteroidEnterpriseBridge

Backend bridge plugin for [Asteroid Enterprise](https://builtbybit.com/resources/84657/). Install this on your Spigot/Paper servers to enable teleportation and command execution for fake players.

## Requirements

- Spigot/Paper 1.8+
- Java 8+
- Asteroid Enterprise on your proxy (Velocity/BungeeCord)

## Installation

1. Download the latest release from [Releases](https://github.com/Asteroid-Enterprise/AsteroidEnterpriseBridge/releases)
2. Drop the JAR into your backend server's `plugins` folder
3. Restart the server

## Features

- **Teleport Players** - Allows proxy to teleport fake players to specific locations
- **Execute Commands** - Allows proxy to run commands on behalf of fake players
